## 2. Recommended Tech Stack

### **Primary Recommendation: Astro + Tailwind CSS**

**Why Astro:**
- ✅ **Performance:** Zero JavaScript by default, perfect for content-heavy marketing site
- ✅ **SEO-Friendly:** Server-side rendering, excellent for search engine visibility
- ✅ **Static Site Generation:** Fast, secure, easy to deploy (Netlify, Vercel, GitHub Pages)
- ✅ **Developer Experience:** Component-based, supports multiple frameworks if needed
- ✅ **Content Management:** Easy to update content in markdown or components
- ✅ **Modern:** Built for performance and developer productivity

**Why Tailwind CSS:**
- ✅ **Utility-First:** Rapid development with consistent design system
- ✅ **Responsive:** Mobile-first approach built-in
- ✅ **Customizable:** Easy to create custom color palette matching brand
- ✅ **Professional:** Clean, modern aesthetic perfect for corporate sites
- ✅ **Accessibility:** Good contrast utilities and semantic HTML support
- ✅ **File Size:** PurgeCSS removes unused styles for optimal performance

**Additional Technologies:**
- **Form Handling:** Netlify Forms, Formspree, or Web3Forms (for contact form)
- **Icons:** Lucide Icons or Heroicons (professional, consistent icon set)
- **Animations:** Minimal CSS animations or Tailwind's built-in transitions
- **Fonts:** Google Fonts (professional sans-serif like Inter, Roboto, or Open Sans)
- **Image Optimization:** Astro's built-in image optimization
- **Deployment:** Netlify or Vercel (free tier, automatic deployments from Git)

**Alternative Stack (if preferred):**
- **Next.js + Tailwind CSS:** More complex but offers more flexibility for future features
- **Plain HTML/CSS/JS:** Simplest option but less maintainable long-term

## 3. Design Direction

### **Visual Identity**
**Color Palette** (to be refined based on logo):
- **Primary:** Professional blue or green (trust, safety, environment)
- **Secondary:** Complementary accent color
- **Neutral:** Grays for text and backgrounds
- **Accent:** Warning orange/yellow for CTAs (safety industry standard)
- **Success:** Green for positive indicators

**Typography:**
- **Headings:** Bold, professional sans-serif (e.g., Inter Bold, Roboto Bold)
- **Body:** Clean, readable sans-serif (e.g., Inter Regular, Open Sans)
- **Size Scale:** Clear hierarchy for accessibility

**Design Principles:**
- ✅ **Professional & Trustworthy:** Clean layouts, ample whitespace
- ✅ **Credibility-Focused:** Certifications, client logos, team credentials prominent
- ✅ **Accessible:** WCAG 2.1 AA compliance minimum
  - Strong color contrast (4.5:1 for text)
  - Semantic HTML5 elements
  - Keyboard navigation support
  - Alt text for all images
- ✅ **Responsive:** Mobile-first design, works on all devices
- ✅ **Restrained:** No flashy animations, focus on content and credibility
- ✅ **Corporate:** Professional imagery, structured layouts

**UI Components:**
- Clean navigation with clear hierarchy
- Card-based service displays
- Professional team member cards
- Client logo grid
- Testimonial sections (if available)
- Clear CTAs throughout
- Professional contact form
